**Firebase** simplifie l'authentification dans les applications **Angular**.

## Intégration de Firebase

### Installation de Firebase dans Angular

```bash
ng add @angular/fire
```

### Exemple d'authentification

```js
import { AngularFireAuth } from '@angular/fire/compat/auth';

constructor(private afAuth: AngularFireAuth) {}

login(email: string, password: string) {
  this.afAuth.signInWithEmailAndPassword(email, password)
    .then(user => console.log('Connexion réussie', user))
    .catch(err => console.error('Erreur de connexion', err));
}
```

### Erreurs fréquentes

- **Utilisation incorrecte des promesses** : Veillez à bien gérer les erreurs avec <Code>catch()</Code> pour éviter les plantages.
- **Oublier de sécuriser les routes** : Utilisez les guards pour restreindre l'accès aux pages protégées.

## Sécurisation des routes

```js
import { AuthGuard } from "./auth.guard";

const routes: Routes = [
  {
    path: "dashboard",
    component: DashboardComponent,
    canActivate: [AuthGuard],
  },
];
```
