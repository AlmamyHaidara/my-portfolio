**Node.js** et **Express** sont des outils puissants pour créer des **API REST** évolutives. Dans cet article, nous allons explorer la création d'une API REST simple avec Node.js, qui permet de gérer des requêtes HTTP pour connecter un frontend au backend.

## Configuration de base

### Installation des dépendances

```bash

npm init -y
npm install express mongoose dotenv

```

### Exemple de serveur Express

```js
const express = require("express");
const app = express();

app.use(express.json());

app.get("/api/users", (req, res) => {
  res.send({ message: "Liste des utilisateurs" });
});

app.listen(3000, () => {
  console.log("Serveur démarré sur le port 3000");
});
```

### Erreurs à éviter

- **Oublier d'utiliser** <Note> <Code> express.json() </Code> </Note>: Cela peut empêcher votre serveur de lire correctement les données JSON envoyées dans les requêtes.

- **Testez vos API avec Postman** pour vérifier les routes et les réponses avant d'intégrer le frontend.

### Conseils

- **Utilisez** <Code>.env </Code> **pour vos variables d'environnement** comme l'URL de la base de données et les clés secrètes.

- **Testez vos API avec Postman** pour vérifier les routes et les réponses avant d'intégrer le frontend.
