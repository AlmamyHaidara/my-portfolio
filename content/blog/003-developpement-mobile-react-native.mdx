**React Native** et **Expo** permettent de développer des applications mobiles multiplateformes avec **JavaScript**.

## Configuration du projet

### Initialisation d'un projet Expo

```bash
npx create-expo-app myApp
cd myApp
npm start
```

### Exemple de composant de base

```js
import React from "react";
import { Text, View } from "react-native";

export default function App() {
  return (
    <View>
      <Text>Bienvenue sur mon application React Native !</Text>
    </View>
  );
}
```

## Erreurs courantes

- **Erreur "SDK version mismatch"** : Mettez à jour Expo avec <Note> <Code>expo upgrade</Code> </Note> pour synchroniser votre SDK.

- **Utilisation incorrecte des hooks** : Attention à l’utilisation de <Code>useEffect</Code> pour éviter des boucles infinies.

## Conseils pour le développement mobile

- **Utilisez Expo Go** pour tester instantanément votre application sur votre téléphone.
- **Prévoyez des conditions pour les permissions** (appareil photo, localisation) afin d’améliorer l'expérience utilisateur.
