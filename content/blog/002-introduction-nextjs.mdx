**Next.js** est un framework basé sur **React** offrant des fonctionnalités telles que le rendu côté serveur (SSR) et la génération de sites statiques.

## Exemple de page Next.js

```js
// pages/index.js
export default function Home() {
  return <h1>Bienvenue sur mon site Next.js !</h1>;
}
```

## Rendu côté serveur (SSR)

Le SSR dans Next.js permet d’améliorer le SEO et les performances.

```js
export async function getServerSideProps() {
  const data = await fetch("https://api.example.com/data");
  return {
    props: {
      data: await data.json(),
    },
  };
}
```

## Erreurs à éviter

- **Oublier d'utiliser** <Code>getServerSideProps</Code> **pour les données dynamiques** : Ne pas le faire peut ralentir le chargement de vos pages.
- **Mauvaise gestion des erreurs d’API** : Toujours prévoir des conditions pour les erreurs réseau ou les API en échec.

## Optimisations

- **Lazy loading** : Améliorez les performances en chargeant les composants au moment où ils sont nécessaires.

```js
import dynamic from "next/dynamic";
const Component = dynamic(() => import("./Component"));
```
