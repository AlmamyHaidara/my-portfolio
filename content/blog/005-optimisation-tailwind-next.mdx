**Tailwind CSS** avec **Next.js** permet de créer des sites rapides et réactifs.

## Utilisation de Tailwind avec Next.js

### Installation

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init
```

### Exemple d'utilisation

```jsx
export default function Home() {
  return (
    <div className="flex justify-center items-center h-screen bg-gray-100">
      <h1 className="text-4xl font-bold">Hello, Tailwind CSS!</h1>
    </div>
  );
}
```

### Erreurs courantes

- **CSS non purgé** : Veillez à purger les classes inutilisées pour réduire la taille des fichiers CSS.

```js
// tailwind.config.js
module.exports = {
  purge: ["./pages/**/*.js", "./components/**/*.js"],
};
```

### Optimisations supplémentaires

- **Lazy loading** des images pour améliorer les temps de chargement.

```js
<img src="/image.jpg" loading="lazy" alt="Exemple" />
```
