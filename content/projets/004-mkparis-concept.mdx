## Présentation

Développement d’une application mobile e-commerce pour MKPARIS Concept, permettant la gestion des ventes de produits de beauté en ligne.

## Missions réalisées

- Développement du front-end mobile avec **React Native** et **TypeScript**
- Création du backend avec **Node.js**
- Intégration de la base de données et du déploiement sur **Azure**
- Livraison d’une solution mobile intuitive pour la gestion des ventes

## Stack technique

- React Native
- TypeScript
- Node.js
- Azure (base de données, déploiement backend)
