**Archive Dossier** est un projet développé pour la société MIDI Sarl. Ce projet permet de gérer efficacement plusieurs aspects de la gestion d'archives, y compris les clients, leurs comptes, ainsi que la gestion des dossiers et des pièces jointes associées. Il propose aussi un système de gestion des documents et des métadonnées dynamiques.

## Fonctionnalités principales

- **Gestion des clients** : Créez, modifiez et gérez les informations des clients et leurs comptes.
- **Gestion des dossiers** : Chaque client peut avoir plusieurs dossiers, regroupant des informations spécifiques.
- **Pièces jointes** : Ajoutez et gérez les pièces jointes associées à chaque dossier.
- **Gestion documentaire** : Gérez les documents associés aux dossiers, y compris le type de document.
- **Métadonnées dynamiques** : Chaque type de document peut avoir des métadonnées spécifiques et dynamiques, permettant une flexibilité accrue dans la gestion des informations.

- <a href="https://nextjs.org/" className="no-underline hover:text-primary font-semibold">Next.js :</a> Framework React pour le développement d'applications web modernes.
- <a href="https://www.typescriptlang.org" className="no-underline hover:text-primary font-semibold">TypeScript (TS) :</a> Langage utilisé pour apporter une typage statique à JavaScript.
- <a href="https://tailwindcss.com/" className="no-underline hover:text-primary font-semibold">Tailwind CSS : </a>Framework CSS pour un design rapide et moderne.
- <a href="https://www.prisma.io/" className="no-underline hover:text-primary font-semibold">Prisma: </a>ORM pour une gestion des bases de données simple et efficace.
- <a href="https://www.postgresql.org" className="no-underline hover:text-primary font-semibold">PostgreSQL: </a> Base de données relationnelle utilisée pour stocker les informations.
- <a href="https://vercel.com/" className="no-underline hover:text-primary font-semibold">Vercel: </a> Plateforme de déploiement pour héberger le projet.
- <a href="https://git-scm.com" className="no-underline hover:text-primary font-semibold">Git: </a> Système de gestion de version pour le suivi du code.
- <a href="https://github.com/" className="no-underline hover:text-primary font-semibold">GitHub: </a> Plateforme de partage et de collaboration autour du code source.

## Objectifs du projet

- Automatiser et optimiser la gestion des archives clients.
- Faciliter l'accès rapide aux informations pertinentes pour chaque dossier.
- Offrir une gestion centralisée des documents et des métadonnées associées.
