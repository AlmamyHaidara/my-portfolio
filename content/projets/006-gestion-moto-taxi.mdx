## Présentation

Application pour la gestion opérationnelle et financière d’une flotte de moto-taxis.

## Missions réalisées

- Développement du front-end avec **Next.js**
- Génération automatique de rapports via **Azure Functions**
- Gestion des données avec **MySQL** et **PostgreSQL**

## Stack technique

- Next.js
- Azure Functions
- MySQL
- PostgreSQL
