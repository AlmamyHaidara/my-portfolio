## Présentation

Développement d’une solution complète pour la gestion des biens immobiliers et des documents associés.

## Missions réalisées

- Développement backend avec **.NET**
- Développement frontend avec **React**
- Mise en place de pipelines CI/CD avec **Azure Pipelines**
- Conteneurisation avec **Docker**

## Stack technique

- .NET
- React
- Azure Pipelines
- Docker
