# Projet Portfolio Personnel

## Présentation

Ce portfolio est construit avec Next.js 14, TypeScript, et Tailwind CSS. Il présente mes projets, compétences, et articles de blog dans une interface moderne et responsive.

## Fonctionnalités

- ⚡️ Next.js 14 avec App Router
- 🎨 Tailwind CSS pour le styling
- 🌙 Mode sombre/clair
- 📱 Design responsive
- 📝 Blog avec MDX
- 🎯 Section projets
- 🛠 Page outils
- 📊 Analytics intégrés
- ⚡️ Performance optimisée

## Stack Technique

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Content:** ContentLayer
- **Animations:** Framer Motion
- **UI Components:** Radix UI
- **Thème:** next-themes
- **Validation:** Zod
- **Date:** date-fns
- **Icons:** Lucide React

## Défis surmontés

- Implémentation d'un système de thème dark/light avec persistance des préférences utilisateur
- Création d'un système de blog avec ContentLayer et MDX
- Optimisation SEO avec métadonnées dynamiques
- Animations fluides avec Framer Motion
- Conception responsive pour tous types d'appareils
